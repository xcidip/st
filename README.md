# **Quick Keybind**
```
# Copy & Paste
Ctrl + Shift + C | Copy Selected
Ctrl + Shift + V | Paste

# Zoom
Ctrl + + | Zoom In
Ctrl + - | Zoom Out
Ctrl + 0 | Reset Zoom

# Scroll
Shift + Page_Up | Scroll Up
Shift + Page_Down | Scroll Down
```

# Requirements
+ Arch
```
yay -S libxft libxinerama ttf-jetbrains-mono-nerd
```

# Install
**Clone repo**
```
git clone https://github.com/xcidip/st.git
```
**Build & Install**

```
./stclean
```
